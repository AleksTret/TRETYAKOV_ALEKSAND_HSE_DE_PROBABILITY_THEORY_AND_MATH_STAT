# TRETYAKOV_ALEKSAND_HSE_DE_PROBABILITY_THEORY_AND_MATH_STAT
Домашние задания по предмету Теория вероятности и математическая статистика в ВШЭ на специальности Инженерии Данных магистранта Третьякова Александра.

Установку можно выполнить при помощи скрипта:

```bash
git clone https://github.com/AleksTret/TRETYAKOV_ALEKSAND_HSE_DE_PROBABILITY_THEORY_AND_MATH_STAT.git

cd TRETYAKOV_ALEKSAND_HSE_DE_PROBABILITY_THEORY_AND_MATH_STAT

(poetry --version) || (pip install poetry)

poetry install --no-root
```
